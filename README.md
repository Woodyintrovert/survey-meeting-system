# Survey Meeting System

## Overview

Survey Meeting System is a web-based attendance survey platform developed using Google Apps Script and Google Sheets as the backend database.

The system supports:

* Meeting creation
* Attendance confirmation (Onsite / Online / Leave)
* Meal requirement survey
* Meeting participant management
* Survey status control (Open / Closed)
* Feedback collection
* Real-time summary reporting

## System Architecture

Google Apps Script serves as the application backend and web service layer.

Google Sheets is used as the database for:

* Committee members
* Meeting information
* Survey responses
* User feedback
